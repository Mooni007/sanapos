📘 Deployment Guide

Firebase Setup: Create project → Enable Firestore → Create collections: books, customers, ledger. Replace firebaseConfig in script (apiKey, projectId...).
HTTPS Required: For camera & Bluetooth, serve via HTTPS (GitHub Pages auto HTTPS).
GitHub Pages: Upload this file as index.html → Settings → Pages → deploy from main branch.
Testing Barcode: Use any product barcode, QuaggaJS will decode.
Bluetooth Print: Ensure printer supports ESC/POS and paired.
